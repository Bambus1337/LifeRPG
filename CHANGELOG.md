# Changelog

All notable changes to LifeRPG will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Next.js 16
- MariaDB database integration with Docker
- User authentication system (placeholder)
- Kickstarter personality quiz
- Dashboard with social features
- Quest system (daily, weekly, monthly)
- Friend suggestion algorithm
- Collaboration system for quests
- Modern UI with green/teal glassmorphism theme
- Responsive mobile design
- Database seed with 10 test users
- Comprehensive documentation
- Docker Compose configuration for MariaDB
- NPM scripts for database management

### Features
- **Authentication:** Login page with placeholder auth
- **Onboarding:** Personality quiz for personalized experience
- **Dashboard:** Stats overview, social scoreboard, quick quest preview
- **Quests Page:** Full quest management with filters and generation
- **Social:** Friend suggestions based on similarity scores
- **Invitations:** Send and accept friend/quest invitations
- **Navbar:** Dynamic navigation with login state awareness

### Database
- 11 tables with proper relationships
- Foreign key constraints
- Indexes for performance
- Test data with Polish content
- MariaDB 11 compatibility

### Documentation
- README with quick start guide
- Setup guide with detailed instructions
- Database schema documentation
- Project summary with architecture
- MariaDB migration guide
- Contributing guidelines
